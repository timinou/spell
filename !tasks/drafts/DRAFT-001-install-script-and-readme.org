* Install Script Design (install.sh)

The script works in 5 phases:

1. *Detect environment* — OS (macOS/Ubuntu/Arch via =/etc/os-release= or =uname=), architecture (x64/arm64). Bail with clear error on unsupported platforms.

2. *Install system dependencies* — only what's needed to build the Rust native crate and run Bun:
   - *Arch*: =pacman -S --needed base-devel git curl= (most Arch users already have these)
   - *Ubuntu*: =apt-get install -y build-essential git curl pkg-config libssl-dev=
   - *macOS*: ensure Xcode CLT (=xcode-select --install=), no Homebrew requirement

3. *Install toolchains* — Bun and Rust, only if not already present:
   - Bun: =curl -fsSL https://bun.sh/install | bash=
   - Rust: =curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y=

4. *Clone and build*:
   - =git clone https://github.com/timinou/spell ~/.spell-src= (or update if exists)
   - =cd ~/.spell-src && bun install=
   - =bun run build:native= (compiles the Rust pi-natives crate)
   - =cd packages/coding-agent && bun link= (makes =spell= available on PATH)

5. *Verify* — run =spell --version= or similar, print success message with next steps.

** UX

- Colored output: green checkmarks for success, red X for failure, cyan headers for phases
- No wall of text — each phase is one line with a spinner or status icon
- Idempotent — re-running updates instead of failing
- Asks for sudo only when needed (package manager installs), explains why
- Usage: =curl -fsSL https://raw.githubusercontent.com/timinou/spell/main/install.sh | bash=

* Install Script (install.sh)

Located at repo root. Invoked via:
=curl -fsSL https://raw.githubusercontent.com/timinou/spell/main/install.sh | bash=

** How it works

5 phases, each printed as a single styled line with status icon:

1. *Detect* — OS via =uname= + =/etc/os-release=, arch via =uname -m=. Supported: macOS (arm64/x64), Ubuntu, Arch. Bail clearly on anything else.

2. *System deps* — install build toolchain only if missing:
   - *Arch*: =sudo pacman -S --needed base-devel git curl=
   - *Ubuntu*: =sudo apt-get update && sudo apt-get install -y build-essential git curl pkg-config libssl-dev=
   - *macOS*: ensure Xcode CLT via =xcode-select -p= check, prompt install if missing

3. *Toolchains* — Bun and Rust, skip if already on PATH:
   - Bun: =curl -fsSL https://bun.sh/install | bash=
   - Rust: =curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y=
   - Source relevant env files after install so they're available for next steps

4. *Clone/update* — =git clone https://github.com/timinou/spell ~/.spell= or =git -C ~/.spell pull --ff-only= if exists

5. *Build* — =cd ~/.spell && bun run build:native && bun run install:dev=. This compiles the Rust native crate and links the =spell= CLI onto PATH.

Final message: print =spell= is ready, suggest =spell --help=.

** UX details

- Colored output: bold cyan phase headers, green =✓= / red =✗= per step
- Idempotent — safe to re-run (updates repo, skips installed deps)
- Asks for sudo only for package manager, explains why
- No spinners (curl-pipe-bash doesn't have a tty reliably) — just clean status lines
- =set -euo pipefail= for safety
