* Scope

Rename the agent tool AND the internal URL protocol from "qml" to "canvas". Covers: tool class, types, file names, prompt, protocol handler, all imports/references. The underlying QML runtime (=@oh-my-pi/pi-qml= bridge, =qml-mode=, =.qml= files) is NOT renamed — only the tool and protocol the LLM/agent sees.

- =qml-mode.ts= / =qml-event-mapper.ts= (desktop mode infrastructure)
- All =.qml= files (UI definitions)
- =packages/qml/= package
- =tools/spell/connect.ts= (uses QmlBridge directly, not the tool)
- =bun-imports.d.ts= =*.qml= module declaration (file extension, not tool name)
